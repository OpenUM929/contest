# 04 — 80줄 분리 규칙

## 적용 기준

- **80줄 초과** → 즉시 폴더 생성 후 내용 재분류
- **80줄 이하** → 단일 파일 유지

## 분리 절차

1. 80줄 초과 감지
2. 주제별로 섹션 그룹핑
3. 폴더 생성 (나침반 문서명 = 폴더명)
4. 각 그룹별로 별도 파일 작성
5. 원본 문서는 나침반으로 전환 (참조 경로만 남김)
6. 00-core.md 분류표 업데이트

## 검증 체크리스트

- [ ] 모든 문서가 80줄 이하인가?
- [ ] 나침반 문서의 경로가 정확한가?
- [ ] 분리 후 원본 문서가 나침반 역할을 하는가?
