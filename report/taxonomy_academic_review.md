# Taxonomy 학술 타당성 검토 (리더십 이론 대조)

> 2026-06-20 · 현재 trait/micro/리더십 유형이 확립된 리더십 연구와 합당한지 검토.
> 기준 프레임워크(검증된 구성개념):
> - **Yukl, Gordon & Taber (2002)** — 리더 행동 위계 taxonomy: **task / relations / change / external** 4개 메타범주 + 12개 구체 행동(clarifying, monitoring, planning, consulting, supporting, recognizing, developing, empowering, visioning, risk-taking, innovating, scanning). *반세기 행동연구의 통합 분류 = "긍정 세트가 완전한가/수준이 맞는가"의 잣대.*
> - **Full Range Leadership Model (Bass & Avolio)** — 변혁(idealized influence·inspirational motivation·intellectual stimulation·individualized consideration)/거래(contingent reward·MBE)/passive-avoidant(laissez-faire).
> - **Ohio State 2요인** — Initiating Structure(과업) vs Consideration(관계): 리더십 실증연구의 가장 견고한 backbone.
> - **부정**: Schmidt(2008) Toxic Leadership Scale 5차원(abusive supervision·authoritarianism·narcissism·self-promotion·unpredictability), Einarsen 파괴적 리더십, Dark Triad(narcissism·Machiavellianism·psychopathy).

---

## 0. 한 줄 결론
- **부정 trait(T101~T106): 학술적으로 매우 견고.** Schmidt 독성리더십 5차원 + FRLM laissez-faire + Dark Triad에 거의 1:1 대응. 다만 **Abusive Supervision(가장 많이 연구된 파괴행동)이 독립 trait로 없고**, self-promotion도 micro로만 존재.
- **긍정 trait(T01~T14): 개별 항목은 대체로 실재하나, 치명적 결함은 "분석 수준(level of analysis) 혼재".** 스타일·역량·상황이 같은 층위에 섞여 있어 개념적으로 겹침 → 사용자가 관찰한 과호출의 *구조적 원인(앵커)* 위에 깔린 *개념적 원인*.
- **누락된 주요 구성개념**: empowering/delegating(위임·임파워먼트), recognizing/contingent-reward(건설적 보상교환), external/boundary-spanning(외부·자원확보) — Yukl 메타범주 중 external이 통째로 비어 있음.

## 1. 긍정 Trait × 학술 구성개념 매핑

| Trait | Yukl 메타범주 | FRLM/기타 대응 | 합당성 판정 |
|---|---|---|---|
| T01 Strategic Decisive | Task(planning/clarifying) | — | △ 행동군은 실재하나 'decisiveness'는 성향(trait)이지 행동군 아님. T06/T10과 과업공간 중복 |
| T02 Collaborative | Relations(consulting) | participative; Consideration | ✓ 견고 |
| T03 Innovative | Change(innovating·risk-taking) | Intellectual Stimulation | ✓ 견고 |
| T04 Crisis Manager | — | — | ✗ **상황(context)이지 trait 아님.** 이미 엔진에 crisis/normal/innovation context_weight 존재 → 맥락 변수로 흡수해야 |
| T05 Analytical | Task(monitoring) 일부 | 합리적 의사결정 스타일 | ✗ **의사결정 스타일**이지 리더십 유형 아님 |
| T06 Execution Driver | Task(clarifying/monitoring) | Contingent Reward 인접 | △ T01·T10과 중복(구조감사 Jaccard로도 확인됨) |
| T07 Coaching | Relations(developing) | Individualized Consideration | ✓ 견고 |
| T08 Emotional Intelligence | — | EI(Goleman) | ✗ **역량/선행요인**이지 'collaborative'와 동렬의 스타일 아님 |
| T09 Visionary | Change(visioning) | Inspirational Motivation+Idealized Influence | ✓ 견고 |
| T10 Strategic Execution | Task | — | ✗ T06+T01의 합성, 독립 근거 약함(이미 중복 trait로 진단됨) |
| T11 Empathetic | Relations(supporting) | Consideration; Servant | ✓ 단 T07/T08과 관계축 인접 |
| T12 Balanced | — | — | ✗ **대응 학술 구성개념 없음.** residual/catch-all. (의도가 exploration-exploitation이면 'ambidextrous leadership'으로 재정의 가능) |
| T13 Ethical Courageous | (integrity) | Ethical Leadership(Brown); Authentic(Walumbwa) | ✓ 견고 |
| T14 Learning Agile | Change(adaptive) | Learning Agility(De Meuse); Adaptive(Heifetz) | ✓ 단 측정 타당도 학계 논쟁 있음 |

### 핵심 결함 — 분석 수준 혼재
긍정 세트는 세 가지 다른 존재론적 층위를 한 줄에 섞었다:
- **(a) 리더십 스타일/행동군** (정당): Collaborative·Visionary·Coaching·Innovative·Empathetic·Ethical
- **(b) 역량/선행요인** (스타일 아님): Emotional Intelligence(T08)·Analytical(T05)·(부분적으로 Learning Agile T14). EI는 리더십을 *예측*하는 능력이지 'collaborative'와 병렬 범주 아님.
- **(c) 상황 한정 역할** (성향 아님): Crisis Manager(T04) = 위기 *맥락*.

→ 한 사람이 여러 trait에 동시 점화되는 이유가 *행동이 달라서가 아니라 범주가 개념적으로 겹쳐서*. 앵커 게이트가 구조적 과호출을 막았다면, 이 수준혼재는 **개념적 과호출**의 잔재.

## 2. 누락된 주요 긍정 구성개념 (Yukl/FRLM 기준 빈자리)
1. **Empowering / Delegating** — Yukl 핵심 행동. 위임·자율부여·의사결정 권한이양. 현재 긍정 micro 빈약한데, 역설적으로 부정 N24(과도통제)·N15-07(과잉보호)는 존재 → *부정 쌍의 긍정 짝이 비어 있음*.
2. **Recognizing / Contingent Reward** — 건설적 성과-보상 교환(거래적 리더십의 효과적 절반). 현재 긍정엔 없고 부정 N09(불공정인정)·N02(보상과장)만 존재.
3. **External / Boundary-spanning / Scanning** — Yukl 4번째 메타범주가 **통째로 부재**. 외부 정보·자원 확보, 이해관계자·네트워킹. 상위 리더일수록 핵심.

## 3. 부정 Trait × 학술 구성개념 (강한 쪽)

| Trait | 대응 검증 구성개념 | 판정 |
|---|---|---|
| T101 Risk: Avoidant | FRLM **Passive-Avoidant / Laissez-faire** | ✓ |
| T102 Risk: Authoritarian | Schmidt **Authoritarianism** | ✓ |
| T103 Risk: Integrity Violation | (anti)Authentic/Ethical Leadership; Einarsen | ✓ |
| T104 Risk: Narcissistic | Schmidt **Narcissism** / Dark Triad | ✓ |
| T105 Risk: Manipulative | Dark Triad **Machiavellianism** | ✓ |
| T106 Risk: Volatile | Schmidt **Unpredictability** | ✓ |

### 부정 쪽 보완점
- **Abusive Supervision 독립 trait 부재**: Tepper의 abusive supervision은 파괴적 리더십에서 가장 많이 연구된 구성개념이자 Schmidt 5차원의 하나인데, 현재는 N15(심리적안전파괴) micro로 흩어져 있고 독립 trait이 없음 → **신설 권고**.
- **Self-promotion**: Schmidt 5차원 중 하나. N08-03(성과 가로채기) micro로만 존재 → trait 승격 검토.

→ 부정 세트는 5/6이 검증된 척도에 직결되어 **긍정보다 이론적으로 더 탄탄**. (이전 PoC에서 부정 F1이 잘 나온 것과도 일관)

## 4. 권고 (우선순위)
1. **상위 조직축 도입 — Yukl 4메타범주**: trait 위에 task/relations/change/external 층을 두면 ① 각 trait이 이론적 소속을 갖고 ② 같은 메타범주 내 trait끼리만 중복 점검하면 되어 과호출 진단이 단순해짐. append-only 구조와 호환(메타 태그만 추가).
2. **수준 혼재 정리**: T05 Analytical·T08 Emotional Intelligence를 *peer trait*에서 **역량/수정자(capability/modifier)**로 재분류(또는 ontological level 태그 부여)해 엔진이 다르게 다루게. T04 Crisis Manager는 **context 변수로 흡수**(엔진에 이미 crisis context 존재).
3. **중복 통합**: T01/T06/T10 과업축 3개 → 의미 기준으로 2개(예: 'Directive/Decisive' + 'Execution')로 정리하거나 앵커로 변별 강화(구조감사와 동일 결론).
4. **빈자리 채우기**: Empowering·Recognizing(contingent reward)·External 긍정 trait/micro 신설 → 평가 데이터 유입 시 1순위 확장 후보.
5. **부정 보완**: Abusive Supervision trait 신설, Self-promotion 승격.
6. **T12 Balanced 재정의 또는 폐기**: 학술 근거 없음. ambidextrous leadership으로 재정의하거나 deprecated.

> 모든 변경은 append-only + `scripts/validate_taxonomy.py` 게이트 통과 원칙 유지([[taxonomy-extension-workflow]]).

## 6. 최신 문헌·신흥 구성개념 (2016~2025 메타분석/종설) — 가장 중요한 교정

고전 토대(Yukl·FRLM·Schmidt) 위에 최근 실증을 얹으면 **결론이 한 방향으로 수렴**한다: *현재 리더십 과학의 핵심 쟁점은 "구성개념 과잉증식(construct proliferation)과 중복(redundancy)"이며, 메타분석 증거는 대부분의 신흥 긍정 구성개념이 변혁적 리더십을 넘어서는 증분설명력이 작다고 말한다.* → 이는 **"긍정 trait을 늘리지 말고 통합하라"**는 방향을 실증으로 지지하고, 내 이전 권고(무작정 신설)를 교정한다.

| 구성개념 | 최신 증거 | taxonomy 함의 |
|---|---|---|
| **Servant 서번트** | Hoch et al. 메타분석: 변혁적 대비 **+12% 증분변량(가장 큼), 상사신뢰 +19%** → *경험적으로 구별됨.* 국내 박동필 메타(588편/839효과크기): **조직효과성에 서번트가 최대 효과크기.** | **유일하게 신설이 정당한 긍정 구성개념.** 현재는 Empathetic(T11)·Coaching(T07)로 파편화 → 통합해 'Servant/Stewardship' trait로 정식화 권고 |
| **Authentic 진성 / Ethical 윤리** | 같은 메타: 변혁적과 **상관 매우 높아 증분 5~11%로 작음, "거의 추가 변량 없음"** = 중복 | **신설 자제 근거.** 현재 T13 Ethical Courageous도 변혁적과 중복 위험 → 별도 trait 유지보다 변혁축(Visionary) 내 속성으로 흡수 검토 |
| **Shared/Distributed 공유** | Wang et al. 메타 ρ=.34(공유 new-genre) vs .19(공유 전통) ; Chamberlin(2024) configuration 연구 | **팀 수준 구성개념**(개인 스타일 아님). 다면평가=개인 진단이므로 **범위 밖**으로 명시(분석수준 경계 결정) |
| **Digital 디지털** | SLR(2025): 4차원 Technological·Visionary-Strategic·Relational-Emotional·Structural-Managerial | 신규 고유 차원은 **Technological/디지털 리터러시뿐**, 나머지는 기존 축과 중복. 현재 T14에 묻힌 M35·M37(디지털·AI)을 **별도 'Digital/Tech' 변별 축**으로 분리 검토(단 과증식 주의) |
| **Identity 정체성**(Haslam ILI 4차원) | 'be one of us/do it for us/craft us/embed us', Steffens et al.(2021) 프로토타입성 메타지지 | **현재 통째 부재 + 변혁적과 구별되는 고유 각도('우리' 관리).** 단 문장단위 micro로 탐지 난이도 높음 → 장기 후보 |
| **Adaptive 적응형**(Heifetz) | 2025 SLR: 특성=유연성·공감·혁신·장기비전 ; 이론 **분열**(Heifetz 실천 vs Uhl-Bien 복잡성), 측정타당도 미정 | 현재 T14(Learning Agile)+T04(Crisis)와 중복. **신설보다 기존 change축 강화로 충분** |

### 수렴 결론
1. **긍정은 통합이 정답**: 메타 증거상 신흥 긍정 구성개념 대부분이 변혁적과 중복 → 내 §1 "분석수준 혼재 = 개념적 과호출" 진단이 **실증으로 확증**됨. 무작정 trait 추가는 오히려 redundancy 악화.
2. **신설이 정당한 단 하나의 긍정 = Servant**(증분변량 최대 + 국내외 메타 일치). 현재 파편(T07·T11)을 통합.
3. **T13(Ethical/Authentic)은 중복 위험** — §1에서 ✓로 봤으나 최신 메타 반영 시 **변혁축 흡수 후보로 하향**.
4. Identity('우리'관리)·Digital(Technological)만 *고유 각도*로 장기 확장 후보. Shared·Adaptive는 각각 *수준 밖*·*기존축 중복*.
5. **부정은 여전히 견고** — 단 Abusive Supervision 신설 권고는 최신 문헌에서도 유효(가장 많이 연구된 파괴 구성개념).

> 종합 우선순위(개정): ①Yukl 4메타축 도입 + 수준혼재 정리(§4-1,2) ②**긍정 통합 우선, 신설 최소화** ③Servant 정식화(파편 통합) ④부정 Abusive Supervision 신설 ⑤Identity·Digital은 데이터·검증 후 장기.
