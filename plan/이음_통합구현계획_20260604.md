**이미 완료된 것:**

- `fetch_folk_museum`: KCISA 문화공공데이터광장 민속아카이브 API(XML) 파싱 완료

- `save_weekly_topic` + `get_active_topic(region)`: 지역 기반 조회, 중앙 폴백 완료
- `WeeklyTopic` 확장: region, welfare_id, question_type, is_customized, parent_topic_id, choices, text_content 등
- `TopicProposal` 확장: question_set_json(JSONB), is_draft, published_topic_id
- `SurveyTemplate`, `SurveyResponse` 신규 테이블
- 프롬프트 4종 파일 존재
- welfare.py: `/generate-questions`, `/refine-questions`, `/validate-questions`, `/save-draft`, `/publish`, `/surveys/*`, `/analytics`, `/responses` 구현
- ✅ `generate_topic_question`: `_generate_with_opencode` 실제 호출, API 키 미설정/실패 시 폴백

**남은 것:**
1. `image` 타입 활성화 (현재 비활성화되어 API 호출 없음)
2. `search_topic_candidates`에서 `audio` 타입에도 공공 API 호출 추가(현재 샘플 폴백)
3. `fetch_gugak`(국립국악원) 함수가 계획서에는 있으나 실제 코드에는 없음

### 3-3. Pipeline C — 감정 분석·심리 지표

**이미 완료된 것:**

- `check_no_contact()`: 3일 이상 미접속 체크
- `get_recent_emotions()`: 최근 N일 감정 이력 조회

**남은 것:**
