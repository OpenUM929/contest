# 03 — 나침반/상세 문서 템플릿

## 나침반 문서 템플릿 (80줄 이하, 경로 목록 + 간단 설명)

```markdown
# {폴더명} GUIDE

## 역할
> 이 폴더의 전체적인 역할을 1-2문장으로 설명

## 하위 경로

| 경로 | 설명 |
|------|------|
| [{하위폴더명1}](하위폴더1/GUIDE.md) | {간단 설명} |
| [{하위폴더명2}](하위폴더2/GUIDE.md) | {간단 설명} |

## 주요 파일

| 파일 | 설명 |
|------|------|
| `{파일명}` | {간단 설명} |
| `{파일명}` | {간단 설명} |
```

## 상세 문서 템플릿 (파일별 기능 + 수정 시 주의사항)

```markdown

## 역할

  - `{함수명}()`: {설명}
- **수정 시 주의사항**:
  - {주의사항}

### {파일명2}
